# Kaushiki Coaching Portal

A comprehensive, full-stack educational SaaS platform designed for Kaushiki Classes. This platform streamlines course management, student progress tracking, fee payments, and communication between faculty, parents, and students.

## 🚀 Project Overview
This platform serves as a centralized hub for managing:
- **Course & Batch Management:** Efficient tracking of students across multiple academic tracks.
- **Role-Based Portals:** Dedicated dashboards for Admin, Faculty, Parents, and Students.
- **Automated Operations:** Integrated payment gateways and SMS notification systems.
- **Performance Tracking:** Real-time progress monitoring and assessment reporting.

## 🛠 Tech Stack
- **Frontend:** Next.js (React), Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (via Prisma ORM)
- **Infrastructure:** Redis (Job Queues), GitHub Actions (CI/CD)

## 📋 Features
- [ ] **Admin Control Center:** Comprehensive CRUD operations and monitoring.
- [ ] **Faculty Portal:** Class scheduling, evaluation, and student feedback tools.
- [ ] **Student Hub:** Progress tracking, assignment management, and streak tracking.
- [ ] **Parent Portal:** Course oversight and progress notifications.
- [ ] **Payment Integration:** Secure checkout and automated billing.

## 🏗 Development Workflow
This project follows an **Agile, frontend-first methodology**. We prioritize UI component development using mocked APIs before moving to backend implementation.
